# Contributing

This project is maintained by Max Dion at AXKI Consulting.

## How to contribute

- Open an issue to report a bug or suggest a feature
- Fork the repo, make your changes, and open a pull request
- All contributions must keep attribution to the original author intact

## Attribution requirements

If you fork or build on this toolkit:

- Keep the `LICENSE` file with the original copyright notice
- Reference AXKI Consulting in your README if publishing publicly
- Do not remove the `CITATION.cff` file
